# 에그 로또 (Egg Lotto)
version 21 위로 표시.

## Features
1. Lotto number recommendation
2. Save the picked number
3. QR code recognizer
4. Winning time reminder
5. Today's horoscope
6. Check the winning result

## Opensource Library
1. Glide
2. Retrofit2
3. Timber
4. Zxing
